# HangmanGame
Ett enkelt spel där man får gissa för att hitta ett hemligt ord. 

Innehåll & Funktioner
**Spelet håller koll på statistik som spelade och vunna matcher. 
**Finns en leaderboard som rankar efter vinstprocent. 
**Finns en funktion för att lägga till nya spelare
**Ord och spelare plus statistik sparas och laddas till och från en JSON-fil.
